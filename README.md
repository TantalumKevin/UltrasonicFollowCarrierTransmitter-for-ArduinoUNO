# Ultrasonic Follow Carrier Transmitter for Arduino UNO

#### 介绍
本项目为2021-2022学年第二学期电气工程学院2020级卓越班嵌入式系统与智能设计课程设计：超声波跟随载物平台之超声发送端。<br>
根据设计，发送端MCU选用```Arduino UNO```，搭配自行设计的外围电路，实现定向发送```40kHz```超声波。
